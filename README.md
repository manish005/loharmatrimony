## 🌟 About The App
Lohar Matrimony is a dedicated and highly interactive community matchmaking application built to bridge the gap for the Lohar community. Focused on privacy, deep cultural alignment, and ease-of-use, it features advanced search parameters, multi-lingual support, real-time messaging, and rigorous KYC verification protocols to ensure a safe environment for finding a life partner.

## ✨ Core Features

### 👤 For Users
* **Multi-Lingual Support:** Seamlessly switch between English, Hindi (हिन्दी), and Marathi (मराठी).
* **Advanced Profile Matching:** Filter partners by Age, Sub-Caste, Education, Income, Country, State, and District.
* **Interactive Dashboard:** Quickly view Recommended Profiles, Shortlisted Matches, and monitor Sent/Received Interests.
* **Real-Time Chat Engine:** Instantly message accepted connections with read-receipts.
* **Secure Document Upload (KYC):** Upload ID documents safely to earn a 'Verified' profile badge, increasing trust.
* **Horoscope & Compatibility:** Share specific astrological details to find highly compatible matches.
* **Help & Support Desk:** Submit support tickets and converse directly with administrators.

### 🛡️ For Administrators (Admin Control Board)
* **Comprehensive User Directory:** View, search, and manage all registered members.
* **KYC Approval Workflow:** Review uploaded documents in a dedicated UI to approve or reject verifications with automated notifications.
* **Subscription Management:** Upgrade users to Silver, Gold, or Platinum premium tiers and set billing durations/expirations.
* **Ticketing System:** Read user issues, provide resolutions, and manage ticket statuses (Open/Resolved/Rejected).
* **Super Admin Privileges:** Exclusive access to the Revenue Dashboard and role-management features.

## 📖 Usage Guidelines

### 1. Account Registration & Onboarding
* Navigate to the home page and click **"Get Started"** or **"Register"**.
* Complete the multi-step onboarding wizard. Provide accurate information to yield the best automated match recommendations.
* Ensure you upload a clear profile photo.

### 2. Getting Verified (Highly Recommended)
* Go to the **KYC Verification** tab inside your dashboard.
* Upload your valid ID documents (Front & Back).
* Wait for an administrator to review your documents. Once approved, you will receive a green **Verified** badge.

### 3. Finding Matches & Interacting
* Use the **Search/Filters** sidebar to drill down into specific criteria (e.g., location, sub-caste).
* Click the **Heart icon** to express Interest, or the **Bookmark icon** to shortlist a profile for later.
* Once an interest is accepted, the **Chat** feature unlocks. Click the Message icon on their profile to start a real-time conversation.

### 4. Admin Navigation
* Administrators log in via the standard route but are automatically granted access to the **Admin Control Board** via the sidebar.
* **Super Admins** have the sole authority to elevate other users to Admin roles and modify subscription records directly.
